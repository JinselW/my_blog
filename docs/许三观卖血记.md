# 许三观卖血记

不愧是余华的文，平静的叙述着所有的苦难。他不评论，似乎只是一个记录者，聆听着每一个角色的自我叙述。

!!! note "回顾下许三观的卖血经历"
    - 第一次，娶了徐玉兰。跟着阿方和根龙，第一次完整地“体验”了卖血。知道卖血前要喝好多好多水，知道卖完血要到饭店去点一盘炒猪肝（生血）和二两黄酒（活血）。
    - 第二次，替一乐赔钱。这一次一乐为了给二乐、三乐撑口气，把方铁匠的大儿子头砸破了。一乐的身世算是在许三观的心里竖起了一道高墙。
    - 第三次，给林芬芳送礼。两人发生关系后的回礼。算是何小勇的事仍在他心里发酵的结果吧。
    - 第四次，闹饥荒，给家人改善生活。大生产的生活过去后，饥荒到来。与一乐之间的高墙开始瓦解（爹你是要带我去吃面条吗？）。
    - 第五次，给一乐、二乐下乡改善生活。
    - 第六次，请二乐队长吃饭。得知阿方膀胱涨破，根龙脑溢血去世。
    - 第七次，一乐肝炎治病的开始。林浦站。他人的温暖。
    - 第八次，百里站。他人的温暖。
    - 第九次，松林站。险些休克。
    - 第十次，黄店站。结识来喜和来顺。
    - 第十一次，七里堡站。
    - 第十二次，长宁站。
    - 第十三次，想吃炒猪肝了。但血站不收。
 没有仔细数之前，感觉好像还没有这么多......十三次啊原来。
  
看到最后，终于看到了余华的气息。和活着一样，不喜不忧。福贵也只是一个人和一头牛在田埂上耕种着，许三观也只是和许玉兰在饭店里吃着炒猪肝，宽慰着。平静的结局，不平静的一生。
但他们也只是那个时代千千万万个相似的人生的缩影吧。年少逝父、饥荒、批斗、生老病死......他们的人生也并没有什么独特的经历，平凡的故事垒叠出了不平凡。每一种人生都有它的百般的滋味。
我们都在用小小的世界观，小小的感官去感受。平凡就是最大的不平凡。

!!! note "这本书中也可以看到许多立体的人物"
    - 许三观：从众（娶了油条西施），好面子（对何小勇这件事很生气，出轨，对一乐的事不管不顾，讨厌许玉兰的大喊大叫，何小勇车祸后串门宣扬），善良（让一乐去给何小勇招魂），爱（在许玉兰被批斗时，守护她；在一家受苦时，想改善生活；将一乐从屋顶上接下，并为其撑腰；在一乐、二乐受苦时，替他们着想；在一乐生病时，不要命了地卖血）......
    - 许玉兰：我觉得是个很正面的角色。要面子（每周不重样的换衣服，何小勇的女人来诉苦时反向炫耀），居安思危（存米），直率，爱（爱一乐，爱许三观，爱这个家），不落井下石，善良......
    - 何小勇的女人，李血头，一乐，二乐，三乐......
  
那样的时代，有人民公社，大运动，饥荒，文革，太多太多的制度黑暗与无力。同样也有那么多的伟大与不屈。他们没有向命运屈服，只是平静地接受，继续着他们的人生。我看到了爸爸的影子，很像吧？里面有句话我很动容，许三观说，我已经体验了很多，但是我儿子的人生才刚刚开始，所以就算我没命了，这辈子也值了。这样为了子女而可以不顾一切的一代人，还会在这个世界停留多久呢？我们又保留了多少呢？我们现在很多人都在抱怨，抱怨他们不是一代合格的父母。可是他们也不知爱是什么，也没有体验过我们想要的爱。又或许我们也不清楚什么是爱以及如何去爱，不是吗？子女和父母本就是一个相互成长的关系吧。我们都在慢慢学着感受爱，去爱。或许他们也已竭尽全力，给了他们认为所能给我们的一切。我们也不应该强求任何人做到我们都不会的事。
